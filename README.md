# Titanic-Advanced-EDA
Titanic Advanced EDA . Performed advanced exploratory data analysis on the Titanic dataset using Python. Handled missing values, created new features like age groups and family size, and used visualizations to uncover deeper survival patterns based on age, embarkation port, and family structure.
## 📌 Overview
Performed advanced exploratory data analysis on Titanic dataset with feature engineering and multiple visualizations.

## 🔧 Key Steps
- Data Cleaning (Handled missing values)
- Feature Engineering (AgeGroup, FamilySize)
- GroupBy Analysis

## 📊 Insights
- Survival varies across age groups
- Embarkation port impacts survival
- Medium family sizes had better survival chances

## 📈 Visualizations
- Histogram (Age)
- Bar plots (AgeGroup, FamilySize)
- Correlation Heatmap

## 🛠 Tools
Python, Pandas, Seaborn, Matplotlib
